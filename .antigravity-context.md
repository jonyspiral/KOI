## Objetivo del proyecto

Reemplazar el servidor **Windows Server 2003 + SQL Server 2000 + PHP 5.2** (`192.168.2.100`) con el **container Docker en Ubuntu** (`192.168.2.210`), que corre PHP 5.6 + MySQL 8.

La BD ya está migrada. El trabajo activo es **validar y arreglar módulos PHP** uno por uno hasta que todo funcione en el Docker.

## Reglas de trabajo

- **Un módulo a la vez**: identificar error → corregir → probar en browser → commit
- **No tocar** código que funciona sin una razón concreta
- **Commit por módulo**: `fix(nombre-modulo): descripción`
- **El servidor `.100` sigue activo** como backup — no apagar hasta validar todo
